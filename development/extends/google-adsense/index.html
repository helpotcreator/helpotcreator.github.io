# GoogleAdSense

# 1단계. 구글 애드센스 사이트 로그인

구글 애드센스 사이트 https://www.google.com/adsenseopen in new window에 접속 후, 시작하기 버튼을 누른다. 구글 로그인도 진행한다.(본 문서에서는 생략함)

# 2단계. 사이트 연결 설정

1단계에서 시작하기 버튼을 눌렀다면, 아래 화면과 같이 사이트 연결 설정 화면이 나타난다. 자신의 웹 사이트 도메인을 적으면 되고, 웹 사이트가 아직 없다면 아직 사이트가 없습니다.에 체크하여 진행하면 된다.

# 3단계. 애드센스 코드 복사하기

위 2단계를 마쳤으면, 자동으로 메인화면으로 이동된다. 우선, 구글 계정 사용자 정보 관련해서 확인한다. 광고는 수익이 발생으로 인한 비용 정산 절차가 있기에 자신의 정보가 확실해야 한다. 애드센스에 사이트 연결 > 시작하기을 누른다.

화면 하단에 보면 애드센스 코드가 잘 생성되었다. 이를 활용하여 자신의 웹 사이트에 연동시키면 된다. 연동 방법은 웹 사이트별로 매우 상이할 것이다. 공식 문서를 참고하여, 자신의 사이트에 맞게 연동시킨다.

# 4단계. 광고 활성화

구글 애드센스를 웹 사이트에 연동 시킨 후, 광고를 활성화하고 심사도 받아야 광고가 시작된다. 아래 화면과 같이 메인 화면에서 탐색을 누른다.

자신이 원하는 형태로 설정들을 임의 조정한다.

# 완료. 심사 진행

사이트 대시보드 화면으로 이동하면, 심사 진행중인 것을 확인할 수 있다. 참고로, 블로그 관련 유명 유튜버 분들에 의하면 블로그에 최소 50개 이상의 포스터가 존재해야 구글 애드센스 광고가 승인된다고 한다.

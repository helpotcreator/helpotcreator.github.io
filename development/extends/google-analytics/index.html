# GoogleAnalytics

자신의 웹 사이트 또는 어플에 구글 애널리틱스를 연동하여 방문자의 모든 활동을 추적할 수 있다. 더군다나, 무료다! 대표적인 기능을 나열하면 아래와 같다.

  • 방문자 관련
    • 국적, 연령, 성별, 관심분야
    • 이용한 기기, 운영체제
  • 통계 관련
    • 기간별 방문자 수
    • 웹 페이지별로 방문자가 머문 시간
    • 웹 페이지 이동한 경로
    • 방문자가 웹 페이지에 머문 시간, 이용한 웹 페이지 순서
  • 등등

# 1단계. 구글 애널리틱스 사이트 로그인

구글 애널리틱스 사이트 https://analytics.google.comopen in new window에 접속 후, 무료로 시작하기 버튼을 누른다. 구글 로그인도 진행한다.(본 문서에서는 생략함)

# 2단계. 계정 및 속성 생성

1단계에서 무료로 시작하기 버튼을 눌렀다면, 아래 화면과 같이 계정 생성 화면이 나타난다. 여기서 말하는 계정은 구글 로그인 계정이 아니다. 구글 애널리틱스를 하나의 인격체로 생각하고, 그것을 만든다고 생각하면 된다. 계정 이름은 자신이 원하는대로 기입한다.

속성 이름도 자신이 원하는대로 기입한다.

계정과 속성의 차이

1개의 계정에 여러 속성을 둘 수 있다. 예를 들어, 자신이 연동시킬 서로 다른 도메인의 웹 페이지를 3개 운영중이라면, 속성을 3개 만들어 연동시키면 된다.

나의 경우는 웹 사이트 연동을 선택했기에, 웹 사이트 URL을 적어야 한다. 스트림 이름은 자신이 원하는대로 기입 후, 스트림 만들기 버튼을 누른다.

스트림이 생성되었고, 이를 잘 복사해서 자신의 웹 사이트 또는 어플에 연동시키면 된다. 연동 방법은 웹 사이트나 어플별로 매우 상이할 것이다. 공식 문서를 참고하여, 자신의 사이트에 맞게 연동시킨다.

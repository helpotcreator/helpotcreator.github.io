# 마우스 커서 애니메이션 효과

# 소개

아래 화면과 같이, 마우스 커서에 애니메이션 효과를 줄 수 있다.

  • 깜박임 효과
  • 이동 효과

# 설정 방법

ctrl + , 버튼을 눌러서 설정을 연다.

  • 위 콤보 박스를 눌러보면 커서 깜박임에 대한 여러 효과를 선택할 수 있는데, 아래 세 가지 중 하나를 추천한다.
    • smooth
    • phase
    • expand
  • 아래 체크 박스에 체크하면 마우스 커서가 이동할 때, 부드러운 애니메이션 효과가 적용된다.

# 구글 스토어 도서 DRM 문서 보안 해제

자신이 구입한 구글 스토어 도서를 자신의 노트북과 핸드폰 등 각종 기기에서 자유롭게 볼 수 있도록 변환 작업입니다.

TIP

이 글을 작성하는 현재(2021.12.10) 사용 가능한 기술입니다.

DANGER

자신이 구입한 파일을 DRM 문서 보안 해제하든 지지고 복든 자신의 마음이지만, 문서 보안 해제된 파일을 다른 사람에게 공유하는 것은 불법입니다.

# 1단계. 프로그램 설치하기

아래 2개 무료 프로그램을 컴퓨터에 설치합니다.

  • Adobe Digital Editionsopen in new window
  • PDF ePub DRM Removalopen in new window

TIP

참고로, PDF ePub DRM Removal 설치할 때 아래와 같은 화면이 뜨면, 계속 무료 버전 사용하겠다는 뜻으로 continue 버튼을 누른다.

# 2단계. 도서 파일 다운받기 .acsm

구글 스토어 문서에서 도서 내보내기를 진행하여, 파일 형태로 다운 받습니다. 확장자가 .acsm인 파일로 다운로드 됩니다.

  • epub 형태로 내보내는 것을 추천 드립니다.
  • pdf 형태의 경우, 3단계 Adobe Digital Editions 프로그램에서 로딩할 때 상하좌우 여백이 사라져 보기 좋지 않은 도서들이 존재합니다.
  • pdf 형태로만 내보내기를 제공하는 도서들도 존재합니다. pdf 형태로도 아래 모든 내용을 동일하게 진행하시면 됩니다.
상세 화면 보기

# 3단계. .acsm -> .epub 변환하기

Adobe Digital Editions 프로그램을 실행하고, .acsm 문서를 읽습니다. .acsm 문서를 읽으면, .epub 형태로 변환되어 아래 폴더에 저장됩니다.

TIP

윈도우 문서 폴더 > My Digital Editions 폴더 > 내부에 .epub 문서

상세 화면 보기

위 작업을 완료하면, Adobe Digital Editions 프로그램은 종료합니다.

# 4단계. .epub 파일의 DRM 문서 보안 해제하기

PDF ePub DRM Removal 프로그램을 실행하여, .epub 파일의 DRM(문서 보안 기술)을 제거합니다. DRM 문서 보안 해제된 파일은 아래 폴더에 저장됩니다.

TIP

윈도우 문서 폴더 > eBook Converter 폴더 > 내부에 .epub 문서

상세 화면 보기

위 작업을 완료하면, PDF ePub DRM Removal 프로그램은 종료합니다. 여기까지 진행하여, 최종 완료된 파일은 어떤 컴퓨터나 핸드폰에서 도서를 볼 수 있습니다. 다만, .epub 파일을 .pdf로 변환하고 싶다면 아래 참고 내용을 추가 진행합니다.
